# Sales & Supply Chain Performance Analysis

## 📌 Project Description
This project focuses on analyzing large-scale sales and logistics data to drive business decisions. I transformed raw transactional records into a structured analytical system to monitor key performance indicators (KPIs) and identify operational bottlenecks.

## 🛠️ Key Technical Features:
* **Advanced Data Cleaning:** Processed over 5,000+ rows of order data, ensuring consistency across categories, dates, and regions.
* **Multi-Dimensional Modeling:** Built complex Pivot Tables to analyze relationships between product categories, customer segments, and regional profits.
* **Logistics Tracking:** Engineered a logic to monitor "On-Time" vs. "Late" shipping performance, providing insights into supply chain efficiency.
* **Interactive Dashboard:** Designed a professional BI dashboard in Excel using Slicers and Dynamic Charts for real-time data exploration.

## 📊 Business Impact:
* **Profitability Insights:** Identified the most and least profitable product sub-categories to help in inventory management.
* **Customer Segmentation:** Provided a deep dive into the behavior of Consumer, Corporate, and Home Office segments.
* **Geographical Analysis:** Visualized sales distribution across different cities and states to support regional marketing strategies.
